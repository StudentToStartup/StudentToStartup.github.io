# Учебная практика — сайт курса

Статический сайт (HTML/CSS/JS). Можно разместить на GitHub Pages.

## Структура
- `index.html` — главная (о курсе, оценивание, календарь, витрина тем, шаблоны)
- `lessons/lesson1.html` … `lessons/lesson8.html` — страницы занятий
- `downloads/` — шаблоны для скачивания (DOCX/XLSX/PPTX)
- `assets/` — изображения и иконки
- `styles.css` — общий стиль
- `js/` — скрипты (витрина тем, чек‑листы прогресса)

## Как опубликовать на GitHub Pages
1. Создайте репозиторий, например `practice-course`.
2. Загрузите содержимое этой папки в корень репозитория.
3. В настройках репозитория откройте **Settings → Pages**.
4. Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)`.
5. Сохраните — через минуту сайт будет доступен.

## Локальный просмотр
Откройте `index.html` в браузере. Для корректной работы скачивания и ссылок лучше использовать простой сервер:

- Python:
  - `python -m http.server 8000`
  - затем откройте `http://localhost:8000`

Чек‑листы на страницах занятий сохраняются в `localStorage` браузера.
